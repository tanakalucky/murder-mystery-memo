# DND機能に@dnd-kit/coreを採用

> 日付: 2026-01-04
> ステータス: 承認済み
> 影響を受けるコンポーネント: MemoList, MemoItem

## コンテキスト

マーダーミステリーメモアプリでは、メモの並び替え機能が必須要件となっている。ユーザーはドラッグ&ドロップ（DND）でメモを自由に並び替え、タイムラインの整理やグルーピングを行う。

要件:

- 直感的なドラッグ&ドロップ操作
- スムーズなアニメーション（60fps）
- 30-40個のメモでも快適に動作
- モダンブラウザ（Chrome メイン）での動作

DND機能を実装するために、適切なライブラリを選定する必要がある。

## 決定

**@dnd-kit/core**を採用する。

追加パッケージ:

- `@dnd-kit/core`: コア機能
- `@dnd-kit/sortable`: ソート可能なリスト
- `@dnd-kit/utilities`: ユーティリティ

## 理由

### メリット

1. **モダンで軽量**
   - React DnDより軽量（バンドルサイズ削減）
   - モダンなReact Hooks API
   - 関数コンポーネントとの親和性が高い

2. **パフォーマンス**
   - GPU アクセラレーション（transform: translate3d()）
   - 仮想化対応（今回は不要だが将来の拡張に対応）
   - 60fps のスムーズなアニメーション

3. **TypeScript完全対応**
   - 型定義が充実
   - 型安全な実装が可能

4. **アクセシビリティ**
   - キーボード操作対応（今回は不要だが良い設計）
   - スクリーンリーダー対応

5. **メンテナンス状況**
   - アクティブに開発されている
   - react-beautiful-dndの後継的存在
   - コミュニティが活発

### デメリット

1. **学習コスト**
   - 新しいライブラリのため、ドキュメントを読む必要がある
   - ただし、API はシンプルで理解しやすい

2. **オーバースペック**
   - 今回は単純なリストのソートのみ
   - 複雑な機能（マルチドロップ、ネストなど）は不要
   - ただし、将来の拡張性は確保される

### 代替案との比較

| 項目             | @dnd-kit    | React DnD   | react-beautiful-dnd | 自前実装 |
| ---------------- | ----------- | ----------- | ------------------- | -------- |
| バンドルサイズ   | 小（~25KB） | 中（~45KB） | 中（~35KB）         | 最小     |
| パフォーマンス   | 高（GPU）   | 中          | 高                  | 可変     |
| TypeScript対応   | 完全        | 完全        | 部分的              | 完全     |
| 学習コスト       | 低          | 高          | 低                  | 最高     |
| メンテナンス     | 活発        | 活発        | 停止中              | 不要     |
| アクセシビリティ | 対応        | 部分的      | 対応                | 未対応   |
| 総合評価         | ⭐⭐⭐⭐⭐  | ⭐⭐⭐      | ⭐⭐⭐              | ⭐⭐     |

**却下理由**:

- **React DnD**: 重く、学習コストが高い
- **react-beautiful-dnd**: メンテナンス停止中（非推奨）
- **自前実装**: 時間がかかり、エッジケースの処理が困難

## 影響

### アーキテクチャへの影響

- `features/memo-list/`と`features/memo-item/`で@dnd-kitを使用
- 状態管理（useMemos）に`reorderMemos`関数を追加
- DND中の状態を管理する必要がある

### 開発への影響

- 依存関係の追加（3パッケージ）
- DND実装の学習コスト（1-2時間程度）
- テストの追加（DND操作のテスト）

### 運用への影響

- バンドルサイズの増加（~25KB gzip後）
- パフォーマンスへの影響は軽微（むしろ改善）
- ブラウザ互換性は問題なし（モダンブラウザのみ対応）

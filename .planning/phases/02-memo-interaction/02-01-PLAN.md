Define the MemoNode types:

```typescript
import type { Node } from "@xyflow/react";

export interface MemoNodeData {
  content: string;
  isEditing: boolean;
}

export type MemoNode = Node<MemoNodeData, "memo">;
```

**Step 2: Create `src/features/canvas/ui/MemoNode.tsx`**

Build the custom node component with these exact behaviors:

1. **Display mode** (`data.isEditing === false`): Render a `<div>` with class `"w-48 rounded border border-black bg-white p-2"` containing a `<p>` with class `"whitespace-pre-wrap text-black"`. If `data.content` is empty string, render a single space `" "` to maintain card height (per D-03, empty memos stay visible).

2. **Edit mode** (`data.isEditing === true`): Render a `<div>` with class `"nodrag w-48 rounded border border-black bg-white p-2"` (note `nodrag` class per D-08 to disable drag while editing). Inside, render a `<textarea>` with:
   - `ref={textareaRef}` for auto-focus
   - `value={data.content}`
   - `placeholder="メモを入力..."` (per D-06)
   - `className="w-full resize-none overflow-hidden bg-transparent text-black outline-none"`
   - `onChange` handler: call `updateNodeData(id, { content: e.target.value })`
   - `onKeyDown={handleKeyDown}` (see below)
   - `onBlur={handleBlur}` (see below)
   - `rows={1}`

3. **Auto-focus**: `useEffect` that calls `textareaRef.current?.focus()` when `data.isEditing` becomes `true`. Also place cursor at end of text via `textareaRef.current.selectionStart = textareaRef.current.value.length`.

4. **Auto-resize textarea** (per D-05): `useEffect` dependent on `[data.content]` that sets:

   ```
   textareaRef.current.style.height = "auto";
   textareaRef.current.style.height = `${textareaRef.current.scrollHeight}px`;
   ```

5. **Keyboard handling** (`handleKeyDown`):
   - ALWAYS call `e.stopPropagation()` first (per D-14, prevent ReactFlow from receiving key events)
   - If `e.shiftKey && e.key === "Enter"`: call `e.preventDefault()` then `updateNodeData(id, { isEditing: false })` (per D-12, MEMO-04)
   - If `e.key === "Escape"`: call `updateNodeData(id, { isEditing: false })` (per D-12, MEMO-05 -- do NOT discard the memo, just exit edit mode)
   - Otherwise: do nothing (Enter inserts newline naturally per MEMO-03)

6. **Blur handling** (`handleBlur`): call `updateNodeData(id, { isEditing: false })` (per MEMO-07, clicking outside confirms edit)

Use `useReactFlow` to get `updateNodeData`. Import `MemoNode` type from `../model/memo-node` for `NodeProps<MemoNode>` typing.

All functions (`handleKeyDown`, `handleBlur`) should be wrapped in `useCallback` with appropriate dependency arrays `[id, updateNodeData]`.

Use `const` + arrow function style per TypeScript coding convention. Named exports only.
</action>

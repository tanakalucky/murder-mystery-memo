## Decisions Made

- Used `forwardRef` + `useImperativeHandle` instead of onReset callback prop because CanvasPage lives outside the ReactFlowProvider scope and cannot call `setNodes` directly. The imperative API keeps reset logic encapsulated in MemoCanvas where `setNodes` is available.
- Used `render={<Button variant="ghost" size="sm" />}` on `AlertDialogTrigger` instead of `asChild` because `@base-ui/react` uses the `render` prop pattern, not Radix-style `asChild`.
- Fixed `forwardRef<MemoCanvasHandle, object>` instead of `Record<string, never>` since `Record<string, never>` has an index signature mapping all keys to `never`, making the inherited `ref` prop incompatible.

**Constants (module-level, before component):**

```typescript
const CARD_HALF_WIDTH = 96; // half of w-48 (192px)
const CARD_HALF_HEIGHT = 20; // approximate half of initial card height
const OVERLAP_THRESHOLD = 20; // px distance to detect same-position creation (per D-11)
const OVERLAP_OFFSET = 20; // px offset for overlapping memos (per D-11)
```

**Imports:**

- `ReactFlow`, `useNodesState`, `useReactFlow` from `"@xyflow/react"`
- `useCallback`, `useRef` from `"react"`
- `NODE_TYPES` from `"../model/node-types"`
- `type MemoNode` from `"../model/memo-node"`

**Component body (`export const MemoCanvas`):**

1. **State**: `const [nodes, setNodes, onNodesChange] = useNodesState<MemoNode>([]);`

2. **ReactFlow instance**: `const { screenToFlowPosition, updateNodeData } = useReactFlow<MemoNode>();`

3. **Last created position ref** (for D-11 overlap detection): `const lastCreatedPosition = useRef<{ x: number; y: number } | null>(null);`

4. **handlePaneDoubleClick** (`useCallback`):
   - Signature: `(event: React.MouseEvent) => void`
   - First, check target is pane: `const target = event.target as HTMLElement; if (!target.classList.contains("react-flow__pane")) return;`
   - Convert coordinates: `const flowPosition = screenToFlowPosition({ x: event.clientX, y: event.clientY });`
   - Calculate memo position (D-09, center on click): `let position = { x: flowPosition.x - CARD_HALF_WIDTH, y: flowPosition.y - CARD_HALF_HEIGHT };`
   - Apply D-11 overlap offset: check if `lastCreatedPosition.current` is not null AND `Math.abs(position.x - lastCreatedPosition.current.x) < OVERLAP_THRESHOLD && Math.abs(position.y - lastCreatedPosition.current.y) < OVERLAP_THRESHOLD`, then add offset: `position = { x: position.x + OVERLAP_OFFSET, y: position.y + OVERLAP_OFFSET };`
   - Update last position ref: `lastCreatedPosition.current = position;`
   - Create new node: `const newNode: MemoNode = { id: crypto.randomUUID(), type: "memo", position, data: { content: "", isEditing: true } };`
   - Add to state: `setNodes((prev) => [...prev, newNode]);`
   - Dependencies: `[screenToFlowPosition, setNodes]`

5. **handleNodeDoubleClick** (`useCallback`):
   - Signature: `(_event: React.MouseEvent, node: MemoNode) => void`
   - Body: `updateNodeData(node.id, { isEditing: true });`
   - Dependencies: `[updateNodeData]`

6. **JSX**: Return `<ReactFlow>` with these props:
   - `nodes={nodes}`
   - `edges={[]}`
   - `nodeTypes={NODE_TYPES}`
   - `onNodesChange={onNodesChange}`
   - `deleteKeyCode={null}` (per D-13, prevent Delete/Backspace from removing nodes)
   - `zoomOnDoubleClick={false}` (prevent zoom on double-click, critical pitfall)
   - `onDoubleClick={handlePaneDoubleClick}`
   - `onNodeDoubleClick={handleNodeDoubleClick}`

Keep the existing comments about panOnDrag and zoomOnScroll defaults if desired, or remove them for clarity.
